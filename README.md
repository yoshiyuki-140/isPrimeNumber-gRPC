# isprimeNumber-gRPC-go


## Install dependencies

```bash
# install grpc
go get -u google.golang.org/grpc
# install protoc's go plugin
go get -u github.com/golang/protobuf/protoc-gen-go
```

## Usage

```bash
go run server.go
```
